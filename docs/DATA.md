# Data Handling Policy

## Data Collection

### User Data
- Username and email for account management
- Chess game statistics (ELO rating, games played)
- Learning progress (lessons completed, puzzle solutions)
- User preferences and settings

### Game Data
- Move history and game outcomes
- Performance analytics and skill assessments
- AI interaction logs for improvement

## Data Processing

### Real-time Processing
- Move validation and game state management
- AI-powered move analysis and hints
- Progress tracking and skill assessment

### Analytics
- Game performance trends
- Learning pathway optimization
- Personalized recommendations

## Data Retention

### Active Data
- User accounts: Retained while account is active
- Game history: 2 years for analysis
- Session data: 30 days
- Progress data: Lifetime of account

### Data Deletion
- Account deletion removes all associated data
- Users can request specific data deletion
- Automated cleanup of expired sessions

## Privacy Protection

### Data Minimization
- Only collect necessary information for functionality
- No tracking beyond essential analytics
- Clear consent for all data collection

### User Rights
- Data portability: Export functionality
- Data correction: Update profile information
- Data deletion: Complete account removal
- Transparency: Clear data usage policies

## External Services

### OpenAI Integration
- Chess move data sent for analysis
- No personal information transmitted
- Responses not stored beyond session

### Database Security
- Encrypted connections (SSL/TLS)
- Access controls and authentication
- Regular security updates
